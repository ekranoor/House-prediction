# 🏠 House Price Prediction
This project predicts house prices using Machine Learning techniques. It helps in estimating the price of a house based on various features like , area, number of rooms,Distance_to_city,price etc.
---
## 🚀 Features
- Data preprocessing (handling missing values, encoding, scaling)
- Exploratory Data Analysis (EDA)
- Machine Learning model (Linear Regression / Ridge  Algorithm)
- Model evaluation (R² score)
- Prediction on new data
## 🛠️ Technologies Used
- Python
- NumPy
- Pandas
- Matplotlib
- streamlit
## 📊 Dataset
- Dataset contains features like:
  - Area
  - Bedrooms
  - Age
  - Distance_to_city
  - Price
